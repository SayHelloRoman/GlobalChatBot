# GlobalChatBot

## Что делать чтобы проект заработал?

- Установить ``PostgreSQL``
- Установить ``requirements.txt``
- Настроить ``config.json``
- Запустить ``setting_db.py`` который создаст и настроит таблицу
- Запустить ``bot.py``
